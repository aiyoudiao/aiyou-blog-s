谷歌浏览器升级了，居然把我谷歌登录过的账号记录都给清空掉了，很麻烦，我的chatgpt登录不上去了，而且我还忘记了那些账号。

<!-- more -->

这时候可以尝试创建新的谷歌账号，然后在用户名那一栏里去输入部分的账号，最后它会弹出你这台电脑在这个输入框中输入过的用户名，然后选中一下，最后如果提示已经存在，那就没错了。

```
https://accounts.google.com/signup/v2/webcreateaccount?flowName=GlifWebSignIn&flowEntry=SignUp
```
